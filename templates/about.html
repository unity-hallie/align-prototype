About Cost and Sustainability

This prototype shows verified AI usage only. Costs are computed from the provider’s API response (usage.prompt_tokens, usage.completion_tokens) and current pricing for the selected model. No estimates or token heuristics are used in the UI.

Data source: Reflection MCP logs stored locally under .local_context/cost_logs/. Each session file contains the exact token counts and computed cost.

Sustainability notes are optional and shown only when verified costs exist. If enabled, comparisons use configurable baselines:

  • EMAIL_COST_USD (default 0.00004)
  • PRINT_PAGE_COST_USD (default 0.05)

These are cost proxies, not energy/CO2 calculations. Disable comparisons if baselines are not maintained.

LLM scoring risks & mitigations: view in app · or open docs/reflection_llm_risk_mitigations.md in the repo.
